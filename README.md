Project: Employee Management System
 
Description:
This project is a Java-based Employee Management System connected to a MariaDB database. It allows users to perform CRUD operations such as adding, viewing, updating, and deleting employee records.
 
Technologies Used:
- Java
- JDBC
- MariaDB
- Eclipse IDE
 
How to Run:
1. Open project in Eclipse
2. Run AppDriver.java
3. Use menu options to interact with system
 
A- View Employees
- Search Employee
- Update Employee
- Delete Employee
- Database integration
